ListenLayer - Cloud Data Layer Platform

== Description ==

ListenLayer is the world’s first, and only cloud data layer platform. Use Listeners to create powerful Data Layers that enhance the accuracy and quality of your website tracking & marketing data – all without programming.

This plugin enables and improves certain data layer activity generated by Listeners inside your ListenLayer account. Sign up for a free ListenLayer.com account. If a Listener you are using prompts you to install this plugin in your Wordpress website, you can do so from this plugin screen. 

Documentation and support for this plugin is provided through your ListenLayer.com user account.
